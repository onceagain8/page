第3包 是 20 20 基础环境集成实施服务 一 、项目总体要求 1、系统必须是非常稳定成熟的软件产品，必须提供完整的产品包装、详细的操作使用说明书、帮助文档、安装程序。 2、系统必须是符合国家结构化的、模块化的、稳定可靠的、实用的产品。必须符合《医院信息系统基本功能规范》新版标准的要求。 3、服务器采用 windows2008 Server 及以上版本操作系统，数据库采用 SQL2008 R2及以上版本；客户机支持 windows XP 、WIN7、WIN8、WIN10。 4、▲系统必须与现有体检管理系统数据无缝对接，数据接口的工作主要是需要将体检者的体检项目信息按照体检排队系统的要求进行处理后，再导入到排队系统中。彩超科室的排队还需要单独处理，需要逐一判断彩超检查项目，并需要与彩超，检查室进行特别关联，保证体检者一次进出彩超检查室即可完成所有彩超检查项目。 5、▲系统必须与现有体检自助登记系统实现数据对接，在自助登记系统完成体检登记，体检者自动进入排检系统，并在指引单打印体检者的第一个排检检查室。 6、系统支持多种体检类型智能排检：健康证，入职体检，驾驶员体检，团体体检在不同区域的排检。 7、系统支持不同区域，不同体检类型的独占检查室，共用检查室排检。 8、系统具备一定的统计分析功能，日志查询功能。 9、本项目与我院其他系统对接，所产生的接口费用，由中标单位支付。
合同履行期限：第一包：供应商为本项目提供自验收合格之日起3年的免费上门维保期。 第二包：供应商为本项目提供自验收合格之日起2年的免费上门维保期。 第三包：供应商为本项目提供自验收合格之日起2年的免费上门维保期。

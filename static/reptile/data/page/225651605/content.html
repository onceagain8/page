天津市海河管理中心 海河窑上口门等堤防险工险段治理工程 (项目编号:GSZC-2020-G-014)竞争性磋商公告)

天津市海河管理中心 海河窑上口门等堤防险工险段治理工程 (项目编号:GSZC-2020-G-014)竞争性磋商公告

项目概况
海河窑上口门等堤防险工险段治理工程 采购项目的潜在供应商应在 天津市南开区桂苑路13号麦迪逊广场2号楼2门302 获取采购文件，并于 2020年11月19日 14点00分 （北京时间）前提交响应文件。
一、项目基本情况
项目编号：GSZC-2020-G-014
项目名称：海河窑上口门等堤防险工险段治理工程
采购方式：竞争性磋商
预算金额：396.05万元
最高限价：396.05万元
采购需求：
包号 是否设置最高限额 预算（万元） 最高限额（万元） 采购目录 采购需求
第1包 是 396.05 396.05 堤坝工程施工 1. 对海河左堤桩号27+444处（窑上口门）现状涵管采用C25微膨胀混凝土进行封堵，封堵长度20m。 2. 对海河左堤桩号27+444处、左堤桩号34+500~34+510段和海河右堤桩号30+475~30+525段、32+235~32+280段、42+700~43+050段、44+650~44+750段共6段挡墙损坏堤段进行挡土墙拆除重建，共计563m。拆除挡墙压顶及500mm高挡墙，500mm以下挡墙损坏部位进行C25混凝土灌注补空。新建挡墙采用C25细石混凝土灌砌石贴坡式挡墙型式，墙高3.3m，墙顶高程3.63m，墙底高程0.33m，顶宽830mm，底板宽2000mm，厚900mm，面坡坡比1:1.1；墙底铺设100mm厚碎石垫层及300g/㎡土工布，墙顶设置100mm厚C25混凝土压顶。 3. 对海河右堤桩号34+260~34+380段破损严重的护脚进行灌砌抛石固脚，长度120m。挡墙迎水侧抛石固脚，抛石底部铺设300g/㎡土工布，抛石顶宽3.0m，顶高程1.60m，迎水侧坡比1:3；抛石上部进行C25混凝土灌砌，厚度500mm。 4. 对海河左堤桩号34+490~34+500段和海河右堤桩号39+030~39+120段两段破损严重护坡及齿脚进行拆除重建，损坏护坡拆除位置为高程2.00m以下，新建护坡采用0.4m厚C25细石混凝土灌砌石，下设0.1m厚碎石垫层，并铺设300g/㎡土工布，护坡坡比1:2.5，齿脚尺寸1.0m（宽）×1.0m（高），齿脚外设抛石固脚。
合同履行期限：25日历天（特殊情况以合同为准）。
本项目不接受联合体参与 ，本项目不接受进口产品

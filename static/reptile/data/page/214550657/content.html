项目概况
天津市和平区南营门街社区卫生服务中心2020年办公用房租赁项目 采购项目的潜在供应商应在 天津市南开区鞍山西道信诚大厦2411 获取采购文件，并于 2020年09月18日 09点30分 （北京时间）前提交响应文件。
一、项目基本情况
项目编号：HCHZC-2020-0901
项目名称：天津市和平区南营门街社区卫生服务中心2020年办公用房租赁项目
采购方式：竞争性磋商
预算金额：34.55万元
最高限价：34.55万元
采购需求：
包号 是否设置最高限额 预算（万元） 最高限额（万元） 采购目录 采购需求
第1包 是 34.55 34.55 房屋租赁服务 1.座落位置：限和平区南营门街行政辖区范围内，交通便利； 2. 租赁面积：建筑面积350平方米以下，租赁后独立使用； 3. 房屋楼层：一层； 4. 水、暖、电、网基础设施配备齐全，能够满足日常医疗需求； 5. 房屋产权及物业管理无纠纷； 6. 投标人提供的租赁房屋为国有产权优先。
合同履行期限：自合同签订之日起12个月（2020年10月1日至2021年9月30日）。
本项目不接受联合体参与 ，本项目不接受进口产品
二、申请人的资格要求：
1.满足《中华人民共和国政府采购法》第二十二条规定；
2.落实政府采购政策需满足的资格要求：项目对小微企业给予6%的价格扣除
3.本项目的特定资格要求：1. 供应商须具有独立的法人资格，提供有效期内的营业执照副本或事业单位法人证书或民办非企业单位登记证书或社会团体法人登记证书或基金会法人登记证书复印件并加盖公章； 2. 供应商须提供房屋权属证明文件，房屋权属无纠纷。（复印件并加盖供应商公章） 3. 供应商须提供经会计师事务所审计的近年度（2018或2019年度）财务审计报告复印件加盖公章或磋商前1个月以内银行出具的资信证明复印件加盖公章； 4. 供应商须提供“依法缴纳税收和社会保险承诺书”，并附有2019年1月至今任意1个月缴纳社会保障资金及依法纳税的有效票据凭证复印件加盖公章； 5.供应商须提供响应文件截止日前3年在经营活动中没有重大违法记录的书面声明（截至提交响应文件截止日成立不足3年的供应商可提供自成立以来无重大违法记录的书面声明）原件并加盖公章； 6.供应商若法定代表人磋商，须提供法定代表人资格证明书和法定代表人身份证复印件并加盖公章；供应商若为被授权人磋商，须提供法定代表人资格证明书及身份证复印件、法人代表授权书（需由法定代表人签字或盖章）和被授权人身份证复印件并加盖公章； 7.本项目不接受联合体参与磋商。

时间： 2020年09月08日 到 2020年09月14日 ，每天上午09:00至12:00，下午14:00至17:00（北京时间，法定节假日除外）
地点：天津市南开区鞍山西道信诚大厦2411
方式：（1）现场获取；（2）报名时须携带供应商营业执照复印件加盖公章、法定代表人授权委托书和被授权人身份证原件；（3）供应商须在《天津市政府采购网》（网址：http://www.tjgp.gov.cn/gys_login.jsp）上完成注册并成为合格供应商。

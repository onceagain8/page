天津市海河管理中心 二道闸电气设备维修及滑触线更换项目 (项目编号:GSZC-2020-G-013)竞争性磋商公告

二道闸电气设备维修及滑触线更换项目 采购项目的潜在供应商应在 天津市南开区桂苑路13号麦迪逊广场2号楼2门302 获取采购文件，并于 2020年11月19日 10点00分 （北京时间）前提交响应文件。
一、项目基本情况
项目编号：GSZC-2020-G-013
项目名称：二道闸电气设备维修及滑触线更换项目
采购方式：竞争性磋商
预算金额：111.23万元
最高限价：111.23万元
采购需求：
包号 是否设置最高限额 预算（万元） 最高限额（万元） 采购目录 采购需求
第1包 是 111.23 111.23 其他水利工程施工 1. 拆除门卫室2台低压配电柜，在配电室安装1台低压进线柜、1台滤波柜和2台馈线柜，新设低压配电柜均采用GCS柜型，柜体尺寸均为800×1000×2200mm，配电柜距侧墙和后墙距离均为1000mm。配电柜后开挖800×600mm电缆沟，采用复合电缆沟盖板。配电室外设置一孔1200×900mm电缆手孔井。 2. 改造后的院区总进线采用1台400A断路器，负责院区所有用电设备配电，重新安装配电箱和漏电保护断路器。拆除原配电装置。 3. 将院外和院内电缆更换为ZR—YJV22—0.6/1kV五芯电缆，建筑内照明、插座电线更换为ZBV—500V三芯铜导线。改造完成后的低压线路采用TN—S接地方式。 4. 将闸区上、下游电动葫芦滑触线，更换为HD—320单极滑触线。
合同履行期限：25日历天（特殊情况以合同为准）。
本项目不接受联合体参与 ，本项目不接受进口产品

天津市中小企业发展促进局机关 天津市中小企业发展促进局第十四届中博会和2017年“创客大赛”会议服务项目 (项目编号:TGPC-2017-B-0062)成交公告

受 天津市中小企业发展促进局机关 委托， 天津市政府采购中心 以 竞争性磋商 方式,对 天津市中小企业发展促进局第十四届中博会和2017年“创客大赛”会议服务项目 实施政府采购。现将成交结果公布如下：

2.项目编号：TGPC-2017-B-0062

会议服务 无 1. 第十四届中博会天津展团参展方案整体策划、特装展台设计（共24个标准展位）预计展区面积为216平方米，展位费预计16.32万元，电费预计不超过2万元，由成交供应商支付展位费、电费，且需要投标人在响应文件中提供展区设计的初步图纸。（假定9*24m展区，进行初步图纸设计） 2. 企业招展：组织我市20家“专、精、特、新”成长型中小企业参展，参展企业必须有较强的技术创新能力，是行业中较为突出的成长型中小企业。由成交供应商协助采购人确定20家参展企业。 3. 参展企业及展品调研：与参展企业沟通参展需求，对参展产品进行考察、测量，由成交供应商协助采购人确定参展产品。 4. 会务服务 （1） 落实天津特装展台及企业展位：成交供应商需与展会组委会对接落实天津特装展台具体位置情况，包括展台位置图、出入展馆路线等；成交供应商需在开展前组织召开参会企业动员会，向参展企业发放参展手册并注明展位位置图、进出馆路线图和时间安排等。 （2） 落实展商及参会企业家接送站、酒店、餐食安排及展馆与酒店间用车（2辆大巴车4天，5辆商务车4天），接待服务用车费用由成交供应商承担。参展企业人员住宿、餐饮等费用由企业自行承担。 （3） 会前、会中、会后与参会企业家联络沟通，做好展商接待（包含必要的接送站、住宿、餐饮、订票、会场接待）及其他服务对接工作 （4） 具体展会工作（特装展台搭建、电子设备安装摆放、会场协调、现场管理、顺序衔接、参观考察，领导巡馆等）。天津特装展台需体现天津特色，着重体现天津中小企业专精特新产品及20家参展企业风采，需为每一家参展企业提供独立的展台及展板，保证每家参展企业有充分的展览展示空间。特装展台由成交供应商聘请专业的设计公司实施搭建，需在组委会要求的布展时间内完成搭建，搭建过程中各项施工符合展馆的技术要求及安全规定。 （二）2017年创客大赛服务要求 1. 制定具体实施方案和工作流程，制定专家评审工作办法和指标体系。 2. 根据2017年“创客大赛”要求，在天津电视台、天津广播电台、北方网、新华网和腾讯等媒体做好大赛宣传。 （1）在天津卫视新闻、新闻频道《新闻这一刻》和《都市报道60分》中播放大赛简介和项目征集令，首播时间18:30-19:30，重播时间23:30-24:30，每天一次，每次不少于1分钟，连续2天播放。 （2）在交通广播《新闻早班车》中插播大赛简介和项目征集令，时间为7:00-8:00，每天一次，每次不少于5分钟，连续3天；滨海广播、新闻广播、交通广播、经济广播创业及经济类直播节目中插播。 （3）北方网、新华网、腾讯网要闻版，版面不少于1500字符，持续7天，宣传大赛简介和项目征集令。 协调各区中小企业工作主管部门、各个示范基地和众创空间，开展参赛企业项目和创业项目征集及确认汇总工作。 3. 提供不少于60人的高级职称专家资源库，专家领域需包含但不限于创投行业、财务管理、技术创新等，组织专家评审和确定推荐项目名单工作，聘请公证机构对评审环节进行公证； 4. 组织评审会至少8次，每次根据实际需要从专家库中随机抽取3名专家； 5. 从专家库中邀请创投行业、财务管理、技术创新等领域专家对推荐项目进行统一辅导，完善项目申报材料，辅导讲解项目答辩要求，提升推荐项目水平，并对重点项目进行专题辅导，辅导次数不少于9次，每次辅导不少于3名专家，每次辅导不少于30分钟。 6. 对参赛项目进行成果宣传、展览展示，组织中介服务机构和上下游企业与项目进行对接，为参赛企业和团体提供展示交流、市场对接平台。（技术咨询、诊断咨询服务、培训服务、法律服务、投融资服务、成果转化等中介机构和项目相关的上下游企业，不少于20家中介机构和40家企业与项目对接和交流，并在天津市中小企业公共服务平台等省市级平台对项目宣传） 7. 提供固定展厅：展厅面积不小于300平米，有专门的展台、展板、多媒体设备、网络、灯光、音响等设备，展览展示时间一个月。展厅必须在天津市行政区域内。 项 1.0 69.5 69.5
